# Конечные автоматы (State Machines)

## Состояния МК

`McStateMachine` — 6 состояний (`StateType`).

```
[Старт]
   │
   ▼
ConnectingWifi ── успех ──► Idle ── Settings ──► Idle
   │                ▲         │                    │
   └── нет SSID ──►SetupWifi─┘                    │
                                                   │
                    потеря WiFi ──► ConnectingWifi ◄┘

OtaUpdating ◄── из любого состояния
```

| Состояние | Класс | Описание |
|-----------|-------|----------|
| `ConnectingWifi` | `ConnectingWifiState` | Подключение к WiFi. Неудача → AP mode → Idle. Нет SSID → SetupWifi |
| `Idle` | `IdleState` | Ожидание. Вкладки датчиков, LED. Кнопка Settings |
| `SetupWifi` | `SetupWifiState` | Первичный ввод SSID/пароля. Skip → Idle, Apply → reboot |
| `Settings` | `SettingsState` | WiFi, Board, OTA, параметры датчиков. Apply → Idle/reboot |
| `Warning` | `WarningState` | Заглушка |
| `OtaUpdating` | `OtaUpdatingState` | Прогресс OTA на дисплее |

### Переходы

| Из | В | Условие |
|----|----|---------|
| ConnectingWifi | Idle | WiFi подключён или создана AP |
| ConnectingWifi | SetupWifi | Нет сохранённых SSID/пароля |
| Idle | Settings | Кнопка Settings |
| Idle | ConnectingWifi | Потеря WiFi |
| Settings | Idle | Apply без изменений |
| Settings | reboot | Изменены WiFi или параметры цепи |
| SetupWifi | Idle | Skip |
| SetupWifi | reboot | Apply |
| Любое | OtaUpdating | Запуск OTA |

## Состояния датчиков INA

`InaStateMachine` — 2 состояния на каждый датчик (`InaStateType`).

```
Idle ── Track/API start ──► Tracking
  ▲                            │
  └──── Stop/API stop ─────────┘
```

| Состояние | Класс | Описание |
|-----------|-------|----------|
| `Idle` | `InaIdleState` | Мгновенные значения: V, mA, mW, mV на шунте |
| `Tracking` | `InaTrackingState` | Накопление mAh/mWh, пики, мин. напряжение |

### Переходы

| Из | В | Условие |
|----|----|---------|
| Idle | Tracking | Кнопка Track или API `start` |
| Tracking | Idle | Кнопка Stop (сохраняет в файл) или API `stop` (без сохранения) |
| Tracking | Idle | Выход из состояния (смена вкладки, настройки) — сохраняет в файл |

### Уведомления

`InaStateMachine::switchState` → `InaStateMachines` → `notifySensorStatusChanged` → LED-индикация, дисплей.

## Взаимодействие

```
McStateMachine
  └── IdleState.handle() → InaStateMachine.handle()

InaStateMachines
  ├── init() — создаёт InaStateMachine для каждого датчика
  ├── getInaStateMachine(addr) / getInaStateMachine(serial)
  ├── isAnySensorBusy() — true если есть Tracking
  └── getAllSensorsInfo() — JSON

CommandsHandler (API)
  ├── start(id) → InaStateMachine.switchState(Tracking)
  ├── stop(id)  → InaStateMachine.switchState(Idle)
  ├── status(id) → getCurrentState().printStatus()
  └── all()     → getAllSensorsInfo()
```
